# RuStore MVP (Android, Jetpack Compose)

## Функциональность

- **Онбординг:**
  - логотип RuStore
  - приветственный текст
  - кнопка перехода к витрине
  - показывается только при первом запуске

- **Витрина приложений:**
  - список приложений с иконкой, названием, описанием и категорией
  - переход в список категорий
  - фильтрация по категории
  - сброс фильтра до “Все приложения”

- **Карточка приложения:**
  - иконка, название, разработчик
  - категория, возрастной рейтинг
  - кнопка «Установить» (заглушка)
  - горизонтальный список скриншотов
  - при клике открывается экран полноэкранных скриншотов
  - кнопка «Назад» для возврата

- **Категории:**
  - список категорий + количество приложений в каждой
  - пункт «Все приложения»

- **Экран полноэкранных скриншотов:**
  - горизонтальный просмотр всех скринов приложения
  - открывается с того скрина, по которому был клик
  - кнопка «Назад» возвращает в карточку

## Технологии

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Coroutines
- In-memory репозиторий данных (FakeAppRepository)

## Как запустить
1. Склонировать репозиторий:
   
   ```bash
   git clone https://github.com/<username>/<repo>.git
   cd <repo>
   
2.Открыть проект в Android Studio (последняя версия).
3. Дождаться Gradle Sync.
4.Запустить конфигурацию app на эмуляторе или устройстве (API 24+).
